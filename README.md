# كُـل - السوق السوداني الأول

> تطبيق سوق إلكتروني مخصص للسودان مع نظام دفع آمن (Escrow) وعمولة 1%.

## هيكل المشروع

```
kull-app/
├── apps/
│   ├── backend/     # Node.js + Express + Prisma (Port 3001)
│   ├── web/         # React + Vite + Tailwind (Port 5173)
│   └── mobile/      # React Native + Expo
└── packages/
    └── shared/      # أنواع TypeScript المشتركة
```

## طريقة التشغيل

### 1. تشغيل الخادم الخلفي
```powershell
cd apps/backend
pnpm run dev
# يعمل على http://localhost:3001
```

### 2. تشغيل الويب
```powershell
cd apps/web
pnpm run dev
# يعمل على http://localhost:5173
```

### 3. تشغيل الموبايل
```powershell
cd apps/mobile
npx expo start
# مسح QR Code بتطبيق Expo Go
```

## مميزات التطبيق
- ✅ تسجيل بالبريد الإلكتروني ورقم الجوال
- ✅ عرض المنتجات مع السعر الإلزامي وخيار الضمان
- ✅ نظام دفع آمن (Escrow) - الأموال محتجزة حتى تأكيد الاستلام
- ✅ عمولة 1% من كل صفقة ناجحة
- ✅ دعم: بنك الخرطوم، فوري/موبايل كاش، تحويل بنكي
- ✅ لوحة إدارة لمراجعة الدفعات وحل النزاعات
- ✅ واجهة عربية RTL كاملة مع دعم الإنجليزية
- ✅ تطبيق موبايل (iOS + Android)

## تدفق عملية الشراء
1. المشتري يختار منتجاً ويضغط "اشتري الآن"
2. يدفع للتطبيق عبر بنك الخرطوم / فوري / تحويل بنكي
3. التطبيق يحتفظ بالمبلغ (Escrow)
4. البائع يتواصل مع المشتري لترتيب التوصيل
5. المشتري يؤكد الاستلام والسلامة
6. التطبيق يحول المبلغ ناقص 1% للبائع
